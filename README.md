# AI News Auto Blogger

An AI-powered automated blogging and email automation workflow built using n8n, GROQ LLM, Gmail API, and News APIs.

## Features

- Fetches latest AI news automatically
- Generates AI blog articles using GROQ LLM
- Sends automated emails using Gmail API
- Fully automated workflow using n8n
- AI-generated content automation pipeline

## Technologies Used

- n8n
- GROQ API
- Gmail API
- HTTP Request APIs
- JSON
- Generative AI / LLMs

## Workflow Architecture

Manual Trigger  
↓  
News API  
↓  
Code Node  
↓  
GROQ AI  
↓  
Gmail Automation

## Project Description

This project automates the process of collecting AI-related news, generating blog articles using Large Language Models, and sending the generated content automatically through Gmail.

The workflow integrates APIs, OAuth authentication, AI content generation, and automation pipelines using n8n.
